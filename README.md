# Job Automation Bot

An AI-powered job discovery platform that aggregates job listings from major employment platforms, featuring intelligent ranking and token-matching search capabilities.

## Features
* **Multi-City Indexing Architecture:** Supports structural data queries across core economic hubs including Bangalore, Mumbai, Delhi, Hyderabad, and Chennai.
* **Token-Based Search Optimization:** Implements case-insensitive token parsing for accurate query matching by role, location, and experience tier.
* **Integrated Profiling Pipeline:** Features direct local document ingestion to validate applicant data vectors against live index matrices.

## Tech Stack
* **Backend Framework:** Python, Flask, Flask-CORS
* **Interface Layer:** HTML5, Tailwind CSS, Editorial Neo-Grotesque Aesthetic Layout
* **Environment Core:** Built-in dynamic data routing structures
